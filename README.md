## 脚本的使用方法：

## 在你的服务器上，执行如下命令即可：

```shell
wget --no-check-certificate https://raw.githubusercontent.com/git2048test/kms-server-deploy/master/kms-server-deploy.sh && bash kms-server-deploy.sh
```
